Zeus Payment Gateway Integration for Woocommerce and Paychoice

== Description ==

<h2>Zeus Payment Gateway Integration</h2>
Zeus Payment Gateway Integration for Woocommerce and Paychoice plugin will link your e-commerce website with <a href="https://sandbox.paychoice.com.au/login.aspx">Paychoice platform</a>. Allowing you to manage concurrent payments from your clients and memberships subscriptions.

Zeus Payment Gateway Integration for Woocommerce and Paychoice make available the functionality of WooCommerce to accept recurrent payments from credit/debit cards using <a href="http://www.paychoice.com.au/">Paychoice Gateway</a>.


== Installation ==

1. Go to the Add New plugins screen in your WordPress admin area
1. Click the upload tab
1. Browse for the plugin file (ZeusBI_PaychoiceWoocommerce.zip)
1. Click Install Now and then activate the plugin
